AI TOKEN MONITOR

Privacy Policy

시행일: 2026-07-03

AI Token Monitor(이하 "본 앱")는 사용자의 컴퓨터에서만 동작하는 로컬 데스크톱 애플리케이션입니다. 본 앱은 어떠한 개인정보도 수집·저장·전송하지 않습니다. 본 문서는 앱이 접근하는 데이터와 그 처리 방식을 투명하게 알리기 위해 작성되었습니다.

1. 수집하는 개인정보

없습니다. 본 앱은 계정, 로그인, 텔레메트리, 분석 도구, 광고 식별자, 쿠키를 일절 사용하지 않으며 개발자를 포함한 어떤 외부 주체에게도 데이터를 전송하지 않습니다.

2. 로컬에서 접근하는 데이터

본 앱은 사용량 통계 계산을 위해 아래 로그 파일을 읽기 전용으로 접근합니다. 이 파일들은 Claude Code와 Codex CLI가 사용자 컴퓨터에 생성한 것으로, 본 앱이 내용을 수정·삭제·업로드하지 않습니다.

대상경로용도
Claude Code 로그~/.claude/projects/**/*.jsonl토큰 수·비용 집계
Codex CLI 로그~/.codex/sessions/**/*.jsonl토큰 수·비용 집계

집계 결과는 메모리에서만 처리되며 별도 파일이나 외부 서버에 저장되지 않습니다. 로그의 대화 내용(프롬프트·응답 텍스트)은 파싱 대상이 아니며, 토큰 수·모델명·타임스탬프 필드만 읽습니다.

3. 네트워크 통신

본 앱이 수행하는 네트워크 요청은 환율 조회 1건이 유일합니다.

  • 대상: open.er-api.com (USD→KRW 환율, 1일 1회 수준)
  • 전송되는 데이터: 없음 — 파라미터 없는 단순 조회(GET) 요청입니다.
  • 요청 실패 시 내장된 고정 환율(1,520원)을 사용하며 앱은 완전히 오프라인으로 동작합니다.
4. 제3자 제공 및 위탁

수집하는 정보가 없으므로 제3자 제공, 처리 위탁, 국외 이전이 발생하지 않습니다.

5. 사용자의 통제권
  • 본 앱은 오픈소스이며 전체 소스 코드를 누구나 검증할 수 있습니다.
  • 앱을 삭제하면 앱과 관련된 모든 것이 제거됩니다. 로그 파일은 각 CLI 도구의 소관이며 본 앱과 무관하게 유지됩니다.
6. 방침 변경 및 문의

본 방침이 변경되는 경우 저장소의 본 문서를 통해 고지합니다. 문의는 GitHub 저장소의 Issues로 부탁드립니다.

Privacy Policy

Effective date: 2026-07-03

AI Token Monitor ("the App") is a local desktop application that runs entirely on your computer. The App does not collect, store, or transmit any personal information. This document transparently describes what data the App touches and how it is handled.

1. Personal Data We Collect

None. The App has no accounts, no sign-in, no telemetry, no analytics, no advertising identifiers, and no cookies. No data is ever sent to the developer or any other party.

2. Data Accessed Locally

To compute usage statistics, the App reads the following log files in read-only mode. These files are created on your machine by Claude Code and the Codex CLI; the App never modifies, deletes, or uploads them.

SourcePathPurpose
Claude Code logs~/.claude/projects/**/*.jsonlToken & cost aggregation
Codex CLI logs~/.codex/sessions/**/*.jsonlToken & cost aggregation

Aggregates are computed in memory only and are never written to files or external servers. Conversation content (prompt/response text) is not parsed — only token counts, model names, and timestamps are read.

3. Network Communication

The App makes exactly one kind of network request: an exchange-rate lookup.

  • Endpoint: open.er-api.com (USD→KRW rate, roughly once per day)
  • Data transmitted: none — a plain parameterless GET request.
  • If the request fails, a built-in fixed rate (₩1,520) is used and the App works fully offline.
4. Third-Party Sharing

Since no data is collected, there is no third-party sharing, processing delegation, or cross-border transfer of any kind.

5. Your Control
  • The App is open source; anyone can audit the full source code.
  • Uninstalling the App removes everything related to it. The log files belong to the respective CLI tools and remain unaffected.
6. Changes & Contact

Any changes to this policy will be announced through this document in the repository. For questions, please open an issue on GitHub Issues.

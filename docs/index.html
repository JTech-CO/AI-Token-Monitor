Claude Code · Codex CLI のトークン使用量とコストを
リアルタイムで表示するデスクトップオーバーレイ

로컬에 기록되는 세션 로그(JSONL)를 직접 파싱해 약 0.5초 내에 갱신됩니다. 모든 데이터는 로컬에서만 처리되며, 개인 사용부터 수 T(조) 토큰 규모의 헤비 유저·팀 사용량까지 같은 화면에서 확인할 수 있습니다.

It parses the session logs (JSONL) written on your machine and updates within about 0.5s. Everything is processed locally — from personal use up to heavy-user and team workloads in the multi-trillion token range, all on a single screen.

ローカルに記録されるセッションログ（JSONL）を直接パースし、約0.5秒以内に 更新されます。すべてのデータはローカルでのみ処理され、個人利用から 数T（兆）トークン規模のヘビーユーザー・チームの使用量まで、 同じ画面で確認できます。

  • 정확한 집계 — 중복 로그 dedup, 캐시 읽기/쓰기 단가 분리 계산
  • 프로바이더 구분 — ALL / CLAUDE / CODEX 탭, 일별 스택 바 차트
  • 인포그래픽 — 활동 히트맵, 모델별 분석(주/월/전체), 캐시 효율 도넛
  • 기간 통계 — 주간 / 월간 / 전체, k · M · B · T 단위
  • 통화 전환 — USD 기본, 실시간 환율 KRW·JPY 전환
  • 트레이 상주 — 컴팩트 모드, 휠 1틱 = 1페이지 스크롤
  • Accurate accounting — log deduplication, separate cache read/write pricing
  • Per-provider view — ALL / CLAUDE / CODEX tabs, daily stacked bar chart
  • Infographics — activity heatmap, per-model breakdown (wk/mo/all), cache efficiency donut
  • Period stats — weekly / monthly / all-time, k · M · B · T units
  • Currency toggle — USD by default, live-rate KRW·JPY conversion
  • Tray resident — compact mode, one wheel tick = one page
  • 正確な集計 — 重複ログの除去（dedup）、キャッシュ読み取り/書き込み単価の分離計算
  • プロバイダー別表示 — ALL / CLAUDE / CODEX タブ、日別スタックバーチャート
  • インフォグラフィック — アクティビティヒートマップ、モデル別分析（週/月/全体）、キャッシュ効率ドーナツ
  • 期間統計 — 週間 / 月間 / 全期間、k · M · B · T 単位
  • 通貨切替 — USD 基本、リアルタイム為替レートで KRW·JPY 換算
  • トレイ常駐 — コンパクトモード、ホイール1ティック = 1ページ
